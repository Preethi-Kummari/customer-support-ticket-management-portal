# 🎫 Customer Support Ticket Management Portal

A responsive Customer Support Ticket Management Portal built using **HTML, CSS, JavaScript, Bootstrap 5, and MockAPI**. The application allows administrators to manage customer support tickets through a clean and modern interface.

---

## 🚀 Features

- Secure Login Page
- Dashboard with Ticket Statistics
- Ticket Analytics Chart
- Create New Ticket
- View Ticket Details
- Edit Existing Tickets
- Delete Tickets
- Search Tickets
- Sort Tickets
- Pagination
- Responsive Design
- Dark / Light Mode
- Settings Page
- Custom 404 Error Page
- Error Handling
- Loading Spinner
- Toast Notifications
- Form Validation
- Network Status Detection
- MockAPI Integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Font Awesome
- Chart.js
- MockAPI
